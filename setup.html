⚠️ CRITICAL: Enable WebRequest in MT5 FIRST

Without this step, the EA cannot communicate with the server and NOTHING will work. Do this before attaching the EA.

⚡ Quick Setup (5 steps)

1

Enable WebRequest in MT5

Open MT5 → Tools → Options → Expert Advisors
✅ Check "Allow WebRequest for listed URL"
Click "+ add new URL" and type: https://script.google.com
Click OK.

2

Install the EA

Download the .ex5 file from your Downloads page.
Copy it to: MT5 → File → Open Data Folder → MQL5 → Experts
Restart MT5.

3

Attach EA to Chart

Drag "FALCON AI" from the Navigator panel onto any chart.
In the settings input fields:
• RC_SecretToken → paste your Secret Token (above)
• RC_EndpointURL → paste the Server URL (above)
Click OK. The EA should show a green smiley face.

4

Open the Dashboard

Download from your Downloads page or open directly:
falconquantai.com/DASHBORED.html
Enter your Secret Token when prompted. Click Connect.

5

Start Trading!

Once connected, the dashboard shows "EA Online" in green. You can now control everything remotely — enable/disable trading, manage symbols, set daily targets, and chat with the AI assistant.
