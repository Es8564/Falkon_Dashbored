Installation Guide

1

Download the EA file

Click the download button above to get the .ex5 file.

2

Copy to MT5 Experts folder

Open MT5 → File → Open Data Folder → MQL5 → Experts. Paste the file there.

3

⚠️ Enable WebRequest (CRITICAL)

In MT5: Tools → Options → Expert Advisors

Check ✅ "Allow WebRequest for listed URL"

Click the "+ add new URL" line and type exactly:

https://script.google.com

⚠️ Without this step, the EA cannot communicate with the server. The dashboard, remote control, and all trading features will NOT work.

4

Attach to a chart

Drag Falcon AI from the Navigator panel onto any chart. Enter your secret token in the settings.

5

Download & install the Dashboard

Download the Dashboard file and save it anywhere on your device. Open it in any browser to connect.

6

Install as app (optional)

On mobile: open in Chrome/Safari → tap "Add to Home Screen". On PC: open in Chrome → click install icon in address bar. This gives you a native app experience.
